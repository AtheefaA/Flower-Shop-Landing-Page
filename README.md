# 🌸 Flower Shop Landing Page

## 📌 Project Overview

The **Flower Shop Landing Page** is a responsive and visually appealing website designed for a flower shop. It showcases beautiful floral arrangements with a modern user interface, elegant layout, and smooth user experience.

This project is built using **HTML, CSS, and JavaScript** and serves as a frontend web development project for learning responsive design and UI creation.

---

## 🎯 Objectives

* Design an attractive flower shop landing page.
* Create a responsive and user-friendly interface.
* Practice HTML, CSS, and JavaScript concepts.
* Build a modern and visually engaging website.

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
